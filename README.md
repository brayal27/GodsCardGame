# Gods Card Game

Proyecto de Trabajo Final de Grado orientado al desarrollo de un videojuego de estrategia por turnos basado en cartas y posicionamiento táctico sobre tablero.

## Descripción

Gods Card Game es un prototipo digital de juego de cartas competitivo para dos jugadores. Cada participante dispone de un mazo de unidades inspiradas en la mitología griega, con atributos de combate y habilidades especiales.

A diferencia de los juegos de cartas tradicionales centrados únicamente en estadísticas y efectos, esta propuesta incorpora un tablero de 4x4 donde la posición de las cartas influye directamente en la estrategia, el ataque y la defensa.

El objetivo del proyecto es diseñar e implementar una experiencia accesible, estratégica y rejugable.

## Características principales

- Combate por turnos para dos jugadores
- Tablero táctico de 4x4 casillas
- Sistema de cartas con atributos:
  - Vida (Might)
  - Ataque
  - Habilidades pasivas
- Posicionamiento estratégico
- Interfaz drag & drop para colocar cartas
- Inspiración visual en fantasía mitológica

3. Importar la carpeta del proyecto.
